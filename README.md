# Interactive 3D AI Roadmap Visualizer

An immersive web-based 3D experience that transforms a traditional AI learning roadmap into an interactive futuristic journey.

Built for the Zen Sky AI Internship Task using React, Vite, and Three.js.

---

# Project Overview

This project visualizes the journey of learning Artificial Intelligence through interactive 3D nodes inside a cinematic environment.

Users can:
- Explore AI milestones visually
- Interact with floating 3D nodes
- Experience smooth camera transitions
- Read detailed information about each AI concept
- Navigate through the AI learning journey interactively

---

# Features

## Interactive 3D Environment
- Futuristic dark-mode aesthetic
- Floating animated AI nodes
- Dynamic lighting and emissive glow effects
- Particle systems and starfield background

## AI Learning Roadmap
Includes milestones such as:
- Data Processing
- Machine Learning
- Neural Networks
- AI Agents
- Automation

## Smooth Camera Animations
- Dynamic camera focusing
- Cinematic transitions between roadmap nodes

## Micro-interactions
- Hover scaling effects
- Glow animations
- Interactive labels
- Smooth UI transitions

## Responsive Design
- Works on desktop and mobile devices
- Dynamic canvas resizing

# Installation

Clone the repository:

```bash
git clone YOUR_GITHUB_LINK
```

Navigate into project folder:

```bash
cd ai-roadmap
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

# Build for Production

```bash
npm run build
```
